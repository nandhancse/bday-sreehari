# bday-sreehari

A premium gym-inspired birthday surprise website for Sreehari, built with only HTML, CSS, and vanilla JavaScript.

## Project Structure

```text
bday-sreehari/
├── assets/
│   ├── fonts/
│   ├── images/
│   └── sounds/
├── css/
│   └── style.css
├── js/
│   └── script.js
├── index.html
└── README.md
```

## How To Run

Open `index.html` in a browser.

The experience includes:

- Beast Mode loading screen
- Cinematic birthday landing screen
- Interactive barbell plate animation
- Gym-themed confetti and glow effects
- Typewriter birthday message
- Vault challenge and riddle
- Reward reveal with rotating creatine tub
- Animated fake achievements

## Asset Placeholders

The site is fully functional without external local assets. Optional files can be added later:

- `assets/sounds/metal-clank-placeholder.mp3`
- `assets/images/`
- `assets/fonts/`

Fonts are loaded from Google Fonts:

- Bebas Neue
- Poppins
